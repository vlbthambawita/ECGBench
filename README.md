# ECGBench

Reproducible ECG Benchmark data from Open access datasets

## Installation

### Using uv (Recommended)

```bash
# Install from source
uv pip install -e .

# Or install from git
uv pip install git+https://github.com/vajira/ECGBench.git
```

### Using pip

```bash
# Install from source
pip install -e .

# Or install from git
pip install git+https://github.com/vajira/ECGBench.git
```

## Development

```bash
# Install with development dependencies
uv pip install -e ".[dev]"
```

## Usage

```python
import ecgbench

# Your code here
```
